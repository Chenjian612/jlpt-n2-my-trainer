# JLPT N2 文法逻辑分组笔记

## 1. 先看句子是在“加信息”还是“反过来”
- 如果是“再加一个优点/缺点” -> 递进
- 如果是“本来以为会怎样，结果却不是” -> 转折 / 反预期
- 如果是“不能因为A就断定B” -> 保留判断

## 2. 递进组
`うえに` / `ばかりか`

### `うえに`
- 意思：而且、再加上
- 语气：平稳递进
- 适合：两个信息并列补充
- 例：
  - このホテルは駅に近い`うえに`、朝食もおいしい。
- 记忆：
  - 普通加一条，用 `うえに`

### `ばかりか`
- 意思：不但……而且……、甚至……
- 语气：更强的递进
- 强提示：
  - `も`
  - `まで`
- 例：
  - 彼は日本語`ばかりか`、英語まで話せる。
- 记忆：
  - 有“连……都……”感觉时，用 `ばかりか`

一句话区分：
- 平稳补充 -> `うえに`
- 升级补充 -> `ばかりか`

## 3. 转折组
`ものの`

### `ものの`
- 意思：虽然……但是……
- 核心语感：
  - 做了A
  - 本来以为会顺着发展
  - 结果却没有
- 例：
  - 何度も説明した`ものの`、相手は納得してくれなかった。
- 读音：
  - 説明（せつめい）
  - 納得（なっとく）
- 记忆：
  - “本来以为会成，结果没成” -> `ものの`

## 4. 反预期组：分成两类
`にしては` / `わりに`

### `にしては`
- 意思：按……来说，算是……；就……而言，出乎意料
- 看什么：
  - 标准
  - 身份
  - 阶段
  - 年龄
  - 时间长度
- 常见前项：
  - 新人（しんじん）
  - 一年目（いちねんめ）
  - 小学生（しょうがくせい）
  - 五歳（ごさい）
  - 二十歳（はたち）
  - 半年（はんとし）
- 例：
  - 彼は入社一年目`にしては`、かなり落ち着いて対応できる。
- 读音：
  - 入社一年目（にゅうしゃ いちねんめ）
  - 落ち着く（おちつく）
  - 対応（たいおう）
- 记忆：
  - 标准型前项 -> `にしては`

### `わりに`
- 意思：按……来看却……；虽然……却……
- 看什么：
  - 客观条件
  - 事实条件
- 常见前项：
  - 値段が高い（ねだん が たかい）
  - 部屋が広い（へや が ひろい）
  - 駅から近い（えき から ちかい）
  - 勉強した（べんきょう した）
- 例：
  - この部屋は駅から近い`わりに`、家賃はそれほど高くない。
- 读音：
  - 家賃（やちん）
- 记忆：
  - 条件型前项 -> `わりに`

一句话区分：
- 年龄/身份/阶段 -> `にしては`
- 价格/大小/距离/努力量这类条件 -> `わりに`

## 5. 保留判断组
`とはかぎらない`

### `とはかぎらない`
- 意思：不一定……；未必……
- 强提示：
  - `からといって`
  - `だからといって`
- 固定记忆块：
  - `Aからといって、Bとはかぎらない`
- 例：
  - 高いからといって、使いやすい`とはかぎらない`。
- 读音：
  - 限る（かぎる）
  - 使いやすい（つかいやすい）
- 记忆：
  - 一看到 `からといって`，优先想 `とはかぎらない`

## 6. 人物侧面组
`ところがある`

### `ところがある`
- 意思：有……的一面；有这种倾向
- 用途：
  - 总结一个人的性格侧面、行为特点
- 例：
  - 彼は忙しいと言いながら、誘えばいつでも来る`ところがある`。
- 读音：
  - 誘う（さそう）
- 记忆：
  - 不是“只不过是”，而是“有这样的一面”

## 7. 最重要的速判表
- 不但A，而且B -> `うえに` / `ばかりか`
- 不但A，而且连B都…… -> `ばかりか`
- 虽然A，但是结果没按预期 -> `ものの`
- 按这个标准来说，出乎意料 -> `にしては`
- 按这个条件看，结果不相称 -> `わりに`
- 不能因为A就断定B -> `とはかぎらない`
- 有这种一面 -> `ところがある`

## 8. 最该死记的 5 个模板
1. `Aからといって、Bとはかぎらない`
2. `Aばかりか、Bまで`
3. `Aうえに、B`
4. `Nにしては、評価が高い`
5. `条件わりに、結果が予想外`

## 9. 最短记忆口诀
- `にしては`：标准型
- `わりに`：条件型
- `ものの`：预期落空
- `うえに`：平稳递进
- `ばかりか`：升级递进
- `とはかぎらない`：别下绝对结论
