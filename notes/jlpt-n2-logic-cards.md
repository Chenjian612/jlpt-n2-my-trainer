# JLPT N2 文法超短背诵卡片

## 1. 递进
### `うえに`
- 意思：而且、再加上
- 关键词：普通补充
- 记忆：平稳递进
- 例：安いうえに、便利だ。

### `ばかりか`
- 意思：不但……而且……、连……都……
- 关键词：`も` / `まで`
- 记忆：升级递进
- 例：日本語ばかりか、英語まで話せる。

## 2. 转折
### `ものの`
- 意思：虽然……但是……
- 关键词：本来以为会顺，结果没顺
- 记忆：预期落空
- 例：説明したものの、伝わらなかった。

## 3. 反预期
### `にしては`
- 意思：按……来说，算是……
- 关键词：标准型
- 常见前项：新人、五歳、半年、一年目、初めて
- 记忆：标准/身份/阶段 -> `にしては`
- 例：新人にしては、落ち着いている。

### `わりに`
- 意思：按……来看却……
- 关键词：条件型
- 常见前项：高い、広い、近い、勉強した、有名だ
- 记忆：条件/事实 -> `わりに`
- 例：高いわりに、おいしくない。

## 4. 保留判断
### `とはかぎらない`
- 意思：不一定……；未必……
- 关键词：`からといって`
- 记忆：别下绝对结论
- 固定背法：`Aからといって、Bとはかぎらない`
- 例：高いからといって、いいとはかぎらない。

## 5. 人物侧面
### `ところがある`
- 意思：有……的一面
- 关键词：人的特点、倾向
- 记忆：不是“仅仅是”，而是“有这种一面”
- 例：子どもっぽいところがある。

## 6. 一眼速判
- 平稳补充 -> `うえに`
- 升级补充 -> `ばかりか`
- 结果没按预期 -> `ものの`
- 标准型前项 -> `にしては`
- 条件型前项 -> `わりに`
- `からといって` -> `とはかぎらない`
- 人的侧面 -> `ところがある`

## 7. 最短口诀
- `にしては`：标准型
- `わりに`：条件型
- `ものの`：预期落空
- `うえに`：平稳递进
- `ばかりか`：升级递进
- `とはかぎらない`：别下绝对结论
